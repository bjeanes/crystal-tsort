# crystal-tsort

Topological sort implementation for Crystal

## Installation

Add it to your `shard.yml`

```yaml
dependencies:
  tsort:
    github: namusyaka/crystal-tsort
    version: ~> 0.0.1
```

## Usage

```crystal
require 'tsort'
```
